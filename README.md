# Project title: Form Exercise with Formik

## Description of Project:
This project allows show a form with 2 fields (email and password) along with a submit button. The form is intended to be used with the Formik library to change the state of each variable, as well as validation and displaying an alert when everything is completed with the submission handler.
Project validations include when any field is empty or when the mail field is not in the mail format.

## How to Run:
To run the project locally, follow the next steps:
1. Download the project.
2. Open the project with your IDE, for example: Visual Studio Code.
3. Install the libraries using the following command in your IDE terminal (npm install).
4. To run the project use the following command (npm start)
5. The browser will automatically open the project in localhost:3000. Otherwise, you can open the project in that URL.

## Roadmap of future improvements:
The future improvements to apply in this project is to create a register form in a different tab with more fields and add CSS style to both forms.
